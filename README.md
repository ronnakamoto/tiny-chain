> Tiny Chain is a tiny blockchain, based on NodeJS and TypeScript that will feature pluggable consensus mechanisms including but not limited to novel Proof Of Arrival(PoAl) and Proof Of Activity(PoAy).

## Install

```sh
yarn install
```

## Run tests

```sh
yarn run test
```
